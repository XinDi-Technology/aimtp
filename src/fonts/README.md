# 字体安装说明

## 字体文件

已安装的字体文件：

### GWM Sans UI（界面和文档字体）
- `GWMSansUI-Regular.woff2` - 常规字重
- `GWMSansUI-Bold.woff2` - 粗体
- `GWMSansUI-Light.woff2` - 轻量字重

### JetBrains Mono（代码字体）
- `JetBrainsMono-Regular.woff2` - 常规字重
- `JetBrainsMono-Bold.woff2` - 粗体
- `JetBrainsMono-Light.woff2` - 轻量字重

### Monaspace Argon（代码字体）
- `MonaspaceArgon-Regular.woff2` - 常规字重
- `MonaspaceArgon-Bold.woff2` - 粗体
- `MonaspaceArgon-Light.woff2` - 轻量字重

## 字体格式

使用的字体格式：
- `.woff2` (Web Open Font Format 2) - 现代、高效的字体格式

## 字体加载

字体文件会在应用启动时自动加载。如果字体加载失败，会在控制台显示警告信息。

## 注意事项

1. 字体文件名称已配置为当前目录中的实际文件名
2. WOFF2 格式提供了最佳的压缩率和性能
3. 代码字体选项已更新为：JetBrains Mono 和 Monaspace Argon
4. 请确保您拥有使用这些字体的合法授权
